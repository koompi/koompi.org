### **Installation**

* Requirement

    **hugo** package

    ```
    $ sudo pacman -Sy hugo
    ```

* Set up this project

    Cloning the repository and preliminary steps

    ```
    $ git clone https://github.com/koompi/koompi.org.git
    $ cd koompi.org
    $ hugo server
    ```

    Open this browser and run http://localhost:1313
